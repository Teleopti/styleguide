By default we use material icons : http://materialdesignicons.com/

4.1Quick symbol library

Here you can find the icons we currently use

  • "mdi-menu" - Trigger a contextual menu
  • "mdi-menu-down" - Indicate sorting in tables
  • "mdi-chevron-left" - Indicate expandable row
  • "mdi-alert" - Used for notices
  • "mdi-alert-circle" - Used for general notifications
  • "mdi-alert-octagon" - Used for errors
  • "mdi-plus" - Add new item
  • "mdi-content-copy" - Duplicate the item
  • "mdi-pencil" - Edit the item
  • "mdi-delete" - Remove the item
  • "mdi-fast-forward" - Sidenav toggle right
  • "mdi-rewind" - Sidenav toggle left
  • "mdi-magnify" - Indicate a search
  • "mdi-filter-outline" - Indicate a filter
  • "mdi-account" - User and account related actions
